Projects

A portfolio of work created during my time at Dev Bootcamp

Created during the Introduction to JavaScript lessons in Week 7, this game allows you experience first-hand the wacky world of Presidential debating without having to spend all of the money to get there. Based on the status of the audience, try and guess the best option to win the crowd over.

The goal here was to create several JavaScript objects with properties and call functions to modify them. While this game does exactly that, the code here is an initial solution and probably very inefficient! I look forward to revisiting this project over the rest of my time at Bootcamp and refactoring it, so I'll have a good comparison of my work between now and the end of the program.

*Disclaimer: Similar to real politics, this game may or may not include deep strategy, a sense of balance, and/or fun.

Always looking forward to adding more projects!
