The Phase 0 Bootcamp Cheat Sheet

How I Learned to Stop Worrying and Love the Syntax

Ruby Commands

Command Line:

ruby [filename].rb

Run a Ruby file directly from the command line.

rspec [filename].rb

Run a rspec file directly from the command line.

irb

Enter the Interactive Ruby shell.

IRB:

load '[filename.rb]'

Loads a Ruby file into the Interactive Ruby Shell that can be interacted with.

Basic Syntax:

puts

Prints information with a line break after the display

gets

Gets an input from the user. Use the chomp method to remove the carriage return at the end of the input.

Math Operators

+: Addition

-: Subtraction

*: Multiplication

/: Division

**: Exponent

%: Modulus

Comparison Operators

==: Check if two statements are equal

!=: Check if two statements are unequal

<: Check is one statement is less than another

>: Check if one statement is greater than another

<=: Check is one statement is less than or equal to another

>=: Check if one statement is greater than or equal to another

<=>: The combined comparison operator returns different values based on which argument is greater. Used for sorting.

Loops

For Loop

Repeat a given loop for a certain range of conditions

Syntax: for [variable] in [expression] do

While Loop

Repeat a given loop indefinitely while a certain condition is true

Syntax: while [conditional] do

Until Loop

Repeat a given loop indefinitely while a certain condition is false

Syntax: while [conditional] do

Methods:

Define Your Own

def method_name(argument1, argument2)

(your code goes here)

end

compact

Removes all nil values from an array

count

Counts all elements in an array. Can also use a code block to count only elements which evaluate to true.

each

The most basic iterative method. Used to iterate over each element in an array.

Usage: array.each do |placeholder_variable|

each_with_index

Same as each but includes the index as a second placeholder variable.

Usage: array.each do |placeholder_variable, index_placeholder|

flatten

Removes all nested arrays from a main array. Can be done destructively.

insert

Can insert new elements at any point in the array.

Usage: insert(index, object)

join

Returns the contents of an array as a string. Can use an optional separator argument to delineate values.

Usage: join('separator')

length

Returns the length of an object (generally strings or arrays)

push

Appends an object as the last element of an array.

Can also be written as '<<'

reverse

When used on a string, returns the string in reverse

When used on an array, returns a new array with the elements reversed

Can be used destructively - reverse!

select

Iterates over an array using a conditional, and deletes any elements that evaluate false

Can be done destructively.

shuffle

Randomizes the positions of elements within an array.

times

Perform a do loop for some number of times specified by the object times is called on.

to_a

Converts an object to an Array

to_f

Converts a number to a Float

to_i

Converts a number to an Integer

to_s

Converts an object to a String
